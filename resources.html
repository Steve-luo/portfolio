Resources

Step 1: Create a GitHub account

Go to github.com/join and create an account with your @saic.edu email address. 

Step 2: Activate your Github Student Developer Pack

Visit education.github.com/pack/offers and click the "Get your pack" button on the right. 

Step 3: Register your domain

Visit education.github.com/pack/offers and find  in the list. 

Click the link in "Get access by connecting your GitHub account on Namecheap." to register an account and get a free .me domain. 

Step 4: Download software

Visit education.github.com/pack/offers and find  Bootstrap Studio in the list. Download the software from their website and activate your free license. 

Visit desktop.github.com and download the GitHub Desktop client. 

Step 5: Design and export your website using Bootstrap Studio

Use bootstrapstudio.io/tutorials to learn the basics on how to use the software. 

Remember, you can almost always find answers on Google. 

Step 6: Create a repository on GitHub

Visit github.com/new to create a new repository. Choose a name like "homepage", "portfolio", etc. 

Set the repo to private if you don't want others to access your source code directly. 

Step 7: Host your website

After creating the repo, click the  button and then click  Open with GitHub Desktop. 

Choose a place to save your files and move your exported website into the folder. Commit and upload the repo. 

Go to click  Settings in your repo. Scroll to the bottom and find GitHub Pages. Click on the  button and choose "main", then save. 

In Custom Domain, put your newly registered domain here. 

Step 8: Change DNS settings

Change the DNS settings in your control panel at ap.www.namecheap.com/domains/list. 

Step 9: Visit your new website!

Go to your domain and check out your new website! 

Additional Resources

If you want a domain not ending in .me, you can spend some money buying a domain on Namecheap. For example, a .com domain costs $7.98 for the first year and $12.98 for annual renewal. 

Bootstrap Studio does have a relatively steep learning curve. If you want to use something simpler, portfolio.adobe.com allows you to connect your custom domain. The only caveat is once you graduate, you'll lose your free Adobe subscription. If you plan to start paying for Adobe software anyway, this solution is not bad. 

If you want an even simpler solution, there are always services like squarespace.com and wix.com. They do tend to be a little expensive though, but cheaper than keep paying for Adobe.
